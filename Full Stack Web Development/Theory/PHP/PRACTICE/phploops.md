# while loop in php

```php
<?php
 $i = 1; 
while ($i < 6) { 
  echo $i;
  $i++;
}

?>
```
# Dowhile loop in php
```php
<?php
 $i = 1; 
 do {
    echo $i;
    $i++;
} while ($i < 6);

?>
```

# for loop in php
```php
<?php 

for ($i = 0; $i < 10; $i++ ) {
  echo $i;
}
?>
```
#foreach loop in php
```php
<?php 

$colors = array("red", "green", "blue", "yellow"); 


foreach ($colors as $x) {
  echo $x;
}

?>
```
